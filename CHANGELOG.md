# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.1] - 2025-12-03

### Fixed

- Trim function implementation.

## [0.3.0] - 2025-12-02

### Added

- Saving logs to file.
- Option to trim whitespaces.
- Left panel with option to move output into input box.

### Fixed

- Saving preset values while program is running.

## [0.2.0] - 2025-11-30

### Added
- Current settings are now saved into config file and loaded after application start.
- Up to three setting presets can be saved and loaded at runtime so the user can easily switch between them.
- Presets are also stored in config file.

### Fixed

- Left trim in REMOVE mode when prefix was not found in the line.

## [0.1.0] - 2025-11-25
### Added
- Initial release.
